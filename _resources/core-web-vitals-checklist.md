Most startup websites lose organic rankings and conversions to the same handful of Core
Web Vitals issues — oversized hero images, render-blocking scripts, and layout shift from
late-loading fonts. This checklist walks through exactly what to check and fix, in order
of impact.

Use the on-page summary below for a quick read, or grab the full PDF for a step-by-step
audit you can run against your own site.

## What's inside

- A 12-point technical checklist ordered by fix impact vs. effort
- Startup-specific fixes that don't require a dev team or a rebuild
